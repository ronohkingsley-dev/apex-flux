🌌 APEX // FLUX v8.0
Tactical Financial Intelligence Engine 

APEX // FLUX is a high-performance, local-first financial monitoring system designed to navigate the "chaos" of campus economics. it's been engineered to transforms raw Safaricom/Airtel transaction data into actionable tactical intelligence—tracking debt, spending velocity, and XP-based financial growth without ever sending your data to the cloud.
🛠️ Core Engineering Modules
1. The Glassmorphism Sentry (Landing & Security)
Data Sovereignty: Zero-cloud architecture. 100% of financial data remains in localStorage.
Dynamic PIN Protocol: Multi-step hashed authentication layer to prevent unauthorized access.
Autonomous Ghost Mode: Self-shielding privacy logic. The UI automatically blurs all financial figures if the tab loses focus or remains idle for 3 minutes.

2. The Red Ledger (Debt Intelligence)
Liability Tracking: Specialized regex-parsing for Fuliza, Okoa Jahazi, and Airtel Credit.
Auto-Reconciliation: Automatically detects debt repayment messages and updates the Red Liability Bar in real-time.
Multi-Carrier Ingest: Native support for both Safaricom and Airtel Money transaction signatures.

3. The Vortex Engine (Analytics)
7-Day Reset Cycle: Every 168 hours, the system archives the current ledger and generates an Efficiency Grade (A–D).
Predictive Fuel Gauge: A dynamic SVG visualization of remaining weekly "Fuel" (Budget) vs. "Burn Rate" (Spending).
Frequency-Aware Tagger: Intelligently categorizes Till/Paybill numbers and only prompts for "Route Naming" on frequent P2P transfers (>3 occurrences).
4. Local Tactical Advisor
Zero-API Intelligence: A built-in diagnostic engine that scans your local metadata.

Context: Offers localized cost-saving alerts (e.g., "Mama Dua Kiosk" for food or "Lurambi Stage" for transport) based on spending patterns.
Heavy Processing Simulation: 1.5s terminal-style delay to ensure data integrity during margin calculation.
